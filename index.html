ℹ️ About ReGreen

ReGreen is an AI-powered sustainability tool that identifies the best areas for reforestation using data on soil type, rainfall, and geographical coordinates. It combines Artificial Intelligence (AI), Geographic Information Systems (GIS), and real-time data analytics to support smart, region-specific tree planting.

Developed by Gloria Muema for the PLP Hackathon.
